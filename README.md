# Loan Origination System

## Overview

The Loan Origination System is a web-based application designed to streamline and automate the end-to-end loan application process. It enables applicants to submit loan requests while allowing authorized personnel to review, verify, approve, or reject applications through a structured workflow.

## Features

- Secure user authentication
- Role-based access control
- Loan application submission
- Document upload and verification
- Loan review and approval workflow
- Application status tracking
- Dashboard for managing applications
- Automated business process workflow

## Technologies Used

- Bonita Studio
- BPMN (Business Process Model and Notation)
- Java
- Groovy
- MySQL
- HTML
- CSS
- JavaScript

## Workflow

1. Applicant submits a loan application.
2. The application is assigned for review.
3. Required information and supporting documents are verified.
4. The application goes through the approval process.
5. A final decision (Approved or Rejected) is made.
6. The applicant is notified of the outcome.

## Purpose

This project was developed to demonstrate the automation of a loan origination process using Business Process Management (BPM). It showcases how workflow automation can improve efficiency, reduce manual tasks, and provide a structured process for handling loan applications.
